<h1>Terrain Engine Guide</h1>
<p>This section will explain how to use the <span class='doc-keyword'>Terrain Engine</span>.  It will cover creation, technical details, and other considerations.  It is broken into the following sections:
</p>
<div class='vspace'></div><h2><a class='wikilink' href='http://docwiki.hq.unity3d.com/index.php?n=Main.terrain-UsingTerrains'>Using Terrains</a></h2>
<p>This section covers the most basic information about using Terrains.  This includes creating Terrains and how to use the new Terrain tools &amp; brushes.
</p>
<div class='vspace'></div><div><a class='wikilink' href='http://docwiki.hq.unity3d.com/index.php?n=Main.terrain-UsingTerrains'><img src='http://docwiki.hq.unity3d.com/uploads/Main/TerrainGuide-usingTerrains.png' alt='' title='' /></a></div>
<div class='vspace'></div><h2><a class='wikilink' href='http://docwiki.hq.unity3d.com/index.php?n=Main.terrain-Height'>Height</a></h2>
<p>This section explains how to use the different tools and brushes that alter the Height of the Terrain.
</p>
<div class='vspace'></div><div><a class='wikilink' href='http://docwiki.hq.unity3d.com/index.php?n=Main.terrain-Height'><img src='http://docwiki.hq.unity3d.com/uploads/Main/TerrainGuide-HeightStepping.png' alt='' title='' /></a></div>
<div class='vspace'></div><h2><a class='wikilink' href='http://docwiki.hq.unity3d.com/index.php?n=Main.terrain-Textures'>Terrain Textures</a></h2>
<p>This section explains how to add, paint and blend Terrain Textures using different brushes.
</p>
<div class='vspace'></div><div><a class='wikilink' href='http://docwiki.hq.unity3d.com/index.php?n=Main.terrain-Textures'><img src='http://docwiki.hq.unity3d.com/uploads/Main/TerrainGuide-TerrainTextures.png' alt='' title='' /></a></div>
<div class='vspace'></div><h2><a class='wikilink' href='http://docwiki.hq.unity3d.com/index.php?n=Main.terrain-Trees'>Trees</a></h2>
<p>This section contains important information for creating your own tree assets.  It also covers adding and painting trees on your Terrain.
</p>
<div class='vspace'></div><div><a class='wikilink' href='http://docwiki.hq.unity3d.com/index.php?n=Main.terrain-Trees'><img src='http://docwiki.hq.unity3d.com/uploads/Main/TerrainGuide-TreesSection.png' alt='' title='' /></a></div>
<div class='vspace'></div><h2><a class='wikilink' href='http://docwiki.hq.unity3d.com/index.php?n=Main.terrain-Grass'>Grass</a></h2>
<p>This section explains how grass works, and how to use it.
</p>
<div class='vspace'></div><div><a class='wikilink' href='http://docwiki.hq.unity3d.com/index.php?n=Main.terrain-Grass'><img src='http://docwiki.hq.unity3d.com/uploads/Main/TerrainGuide-GrassSection.png' alt='' title='' /></a></div>
<div class='vspace'></div><h2><a class='wikilink' href='http://docwiki.hq.unity3d.com/index.php?n=Main.terrain-DetailMeshes'>Detail Meshes</a></h2>
<p>This section explains practical usage for detail meshes like rocks, haystacks, vegetation, etc.
</p>
<div class='vspace'></div><div><a class='wikilink' href='http://docwiki.hq.unity3d.com/index.php?n=Main.terrain-DetailMeshes'><img src='http://docwiki.hq.unity3d.com/uploads/Main/TerrainGuide-DetailMeshesSection.png' alt='' title='' /></a></div>
<p class='vspace'><a name='Lightmapping' id='Lightmapping'></a>
</p><h2><a class='wikilink' href='http://docwiki.hq.unity3d.com/index.php?n=Main.Lightmapping'>Lightmaps</a></h2>
<p>You can lightmap terrains just like any other objects using Unity's built-in lightmapper. See <a class='wikilink' href='http://docwiki.hq.unity3d.com/index.php?n=Main.Lightmapping'>Lightmapping Quickstart</a> for help.
</p>
<div class='vspace'></div><div><a class='wikilink' href='http://docwiki.hq.unity3d.com/index.php?n=Main.Lightmapping'><img src='http://docwiki.hq.unity3d.com/uploads/Main/terrainLightmap.jpg' alt='' title='' /></a></div>
<div class='vspace'></div><h2><a class='wikilink' href='http://docwiki.hq.unity3d.com/index.php?n=Main.terrain-OtherSettings'>Other Settings</a></h2>
<p>This section covers all the other settings associated with Terrains.
</p>
<div class='vspace'></div><h2>Mobile performance note</h2>
<p>Rendering terrain is quite expensive, so terrain engine is not very practical on lower-end mobile devices.
</p>
